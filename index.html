⚙️ Configuração do Banco de Dados

Siga os passos abaixo para conectar o ValesApp ao seu banco de dados Supabase gratuito.

1
Crie uma conta gratuita em supabase.com → clique em "Start your project"
2
Crie um novo projeto (pode ser qualquer nome, ex: "valesapp"). Anote a senha do banco.
3
Abra o SQL Editor (menu lateral) e execute o script abaixo para criar as tabelas:
create table fornecedores ( id uuid default gen_random_uuid() primary key, nome text not null, tel text default '', prazo integer default 0, pgto text default 'PIX', obs text default '', created_at timestamptz default now(), created_by text default '' ); create table vales ( id uuid default gen_random_uuid() primary key, forn_id uuid references fornecedores(id) on delete set null, data date not null, valor_original numeric not null default 0, pgto text default '', descricao text default '', obs text default '', vencimento date not null, status text default 'aberto', pago_em date, pago_by text, valor_pago numeric, pgto_real text, pgto_obs text, comp_data text, comp_name text, comp_type text, created_at timestamptz default now(), created_by text default '' ); create table devolucoes ( id uuid default gen_random_uuid() primary key, vale_id uuid references vales(id) on delete cascade, tipo text not null, data date, valor_abatido numeric default 0, item_original text default '', item_trocado text default '', descricao text default '', created_at timestamptz default now(), created_by text default '' ); create table logs ( id uuid default gen_random_uuid() primary key, ts timestamptz default now(), user_id text, user_name text, type text, action text, details text ); alter table fornecedores disable row level security; alter table vales disable row level security; alter table devolucoes disable row level security; alter table logs disable row level security;
4
Copie as credenciais: vá em Settings → API → copie a "Project URL" e a "anon public" key.
